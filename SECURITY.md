# Security Notes

## Доступ

- Бот работает только для пользователей из `ALLOWED_TELEGRAM_IDS`.
- Запросы от неавторизованных ID отклоняются middleware на входе.

## Секреты

- Никогда не коммитьте `.env`.
- Храните `BOT_TOKEN` и пароли БД в защищённом secret-store, если возможно.
- При утечке токена: немедленно перевыпустить токен в BotFather и обновить `.env`.

## Сеть

- Для long polling публичные входящие порты не нужны (кроме опционального `8080` health).
- Ограничьте доступ к `8080` firewall-правилами (или уберите проброс порта в compose).

## БД и backup

- Бэкапы содержат персональные данные задач, храните volume на защищённом хосте.
- Рекомендуется шифровать off-site копии, если добавите удалённое хранилище.
